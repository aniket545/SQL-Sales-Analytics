# SQL Sales Analytics Project

## Overview
This project analyzes retail sales data using SQL to uncover business insights related to revenue, profitability, customer behavior, and time-based trends.

## Tools Used
- SQLite
- DB Browser for SQLite
- SQL

## Key Analysis Performed
- Overall sales and profit analysis
- Category and region-wise performance
- Loss-making orders and customers
- Customer segmentation and repeat behavior
- Monthly and yearly sales trends

## Key Insights
- The overall business is profitable, though certain orders and customers generate losses.
- A small group of customers contributes disproportionately to total revenue.
- Sales and profitability vary significantly across regions and customer segments.
- Time-based analysis reveals clear seasonal patterns in sales.

## Files
- `sql_queries.sql` — SQL queries used for analysis
- `sales_analysis.db` — SQLite database file
- `data/` — Raw dataset
